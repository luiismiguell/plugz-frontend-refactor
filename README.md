# Plugz Front-End Refactor

Projeto desenvolvido durante estágio com o objetivo de reestruturar o front-end de um sistema legado, servindo como base para o desenvolvimento do site atual da Plugz (plugz.com.br).

## 📌 Sobre a Plugz

A Plugz é uma plataforma que oferece descontos, benefícios e soluções financeiras, conectando usuários a uma rede de parceiros e facilitando a gestão de gastos no dia a dia.

## 📌 Contexto do projeto

O projeto consistiu na adaptação de um sistema previamente existente (código legado), transformando-o em uma estrutura front-end mais organizada, legível e funcional.

Essa reestruturação contribuiu como base para o desenvolvimento da versão atual do site da plataforma, melhorando a apresentação e a experiência do usuário.

## 🚀 Principais atividades

* Reestruturação do front-end a partir de código legado
* Organização e padronização da estrutura de arquivos
* Melhoria da legibilidade e manutenção do código
* Ajustes na interface visando melhor usabilidade
* Aplicação prática de HTML, CSS e JavaScript em ambiente profissional

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## 🎯 Objetivo

Aprimorar a estrutura e usabilidade do sistema, contribuindo para a evolução da interface da plataforma Plugz e aplicando conhecimentos de desenvolvimento front-end em contexto real.
